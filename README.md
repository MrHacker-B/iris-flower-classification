🌸 Iris Flower Classification using Machine Learning

📌 Project Overview

This project implements a Machine Learning Classification Model to predict the species of an Iris flower based on its sepal and petal measurements. The model is trained using the famous Iris Dataset from Kaggle/UCI and achieves excellent classification accuracy using the Random Forest Classifier.

---

📖 Project Description

The Iris dataset contains measurements of three different Iris flower species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

The objective of this project is to classify an Iris flower into one of these three species using supervised machine learning.
